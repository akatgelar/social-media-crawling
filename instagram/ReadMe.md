# Crawling Social Media - Instagram



## Installation

Install python & pip


Install requirement environment

```
pip install -r requirement.txt
```



## Usage

Change cookie 

```json
HEADERS = {
    'accept': '',
    'accept-encoding': 'gzip, deflate, br',
    'accept-language': 'en-US,en;q=0.9,id;q=0.8',
    'user-agent': '',
    'cookie': ''
}
```

Run project

```
python crawler.py
```
